## Intended Users

- Lawyers
- Legal assistants
- Compliance professionals

## Typical Workflow

1. Select input text or files
2. Choose anonymization layer
3. Review preset settings
4. Run anonymization
5. Review findings report
6. Export redacted outputs

---

## Choosing a Layer

- Layer 1: fast and predictable
- Layer 2: higher accuracy
- Layer 3: conservative and market standard

---

## Understanding Confidence

- Lower values increase sensitivity
- Higher values reduce false positives
- Masking is applied when uncertain by default

---

## Reviewing Findings

The findings report shows:
- What was detected
- Why it was detected
- How it was handled

This supports legal review and justification.

---

## Sharing Outputs

Redacted outputs can be shared.
Findings reports should be treated as sensitive.

---

## Best Practices

- Keep presets consistent within a case
- Review uncertain findings
- Escalate protection when in doubt
